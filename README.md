# Airbnb Price Prediction in Athens

## Description

This project was developed for the Regression Models course in the Data Science degree at ISCTE.

The objective was to identify the factors that influence Airbnb listing prices in Athens and build a predictive regression model based on listing characteristics, location, host information, and review activity.

## Dataset

Source: Inside Airbnb
The dataset contains more than 15,000 Airbnb listings located in Athens, Greece.

## Methodology
- Exploratory Data Analysis (EDA)
- Data Cleaning
- Feature Engineering
- Outlier Detection
- Linear Regression Modeling
- Model Diagnostics
- Performance Evaluation

## Technologies
- R
- RMarkdown
- HTML
- Statistical Modeling
- Regression Analysis

## Main Results
- Location was one of the strongest predictors of price.
- Distance to the Acropolis significantly affected listing prices.
- Accommodation type and host profile were important explanatory variables.
- The final model achieved an adjusted R² of approximately 0.38.
